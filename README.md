# Contact Management Application (C#)

A REST API for managing contacts built with ASP.NET Core 8.0.

## Features
- Get all contacts
- Get contact by ID
- Create new contact
- Update contact
- Delete contact
- OpenAPI/Swagger documentation

## Prerequisites
- .NET 8.0 SDK or later
- Docker (for containerization)

## Running Locally

```bash
dotnet restore
dotnet run
